== Description ==


WooCommerce Square Up Payment Gateway helps you to pay with Square at WooCommerce checkout. 
With this plugin you can even manage refunds between WooCommerce and Square. 
Keeping the testing phase in mind we have integrated Sandbox support as well for making things easy for developers. 


== How it works ==

User pay with square at WooCommerce Checkout.
